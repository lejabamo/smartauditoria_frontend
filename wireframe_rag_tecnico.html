Procesamiento HL

Uso de PyPDF2 y LangChain para la extracción de texto estructurado. Se implementa un solapamiento (overlap) del 10% entre chunks para evitar pérdida de contexto semántico.

Modelo Vectorial

Implementación de sentence-transformers con el modelo multilingüe paraphrase-multilingual-MiniLM-L12-v2. Vectorización de alta fidelidad en 384 dimensiones.

Búsqueda Semántica

Cálculo de Similitud Coseno en tiempo real. El sistema recupera los 5 fragmentos más relevantes para inyectar como contexto al generador de justificaciones.
